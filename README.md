# Squadra

A project with separate frontend and backend components. Follow the steps below to set up and run the application.


```
Prerequisites
Ensure the following software is installed on your system:

Node.js (version 14.x or higher)
npm (comes with Node.js)

Project Structure

Squadra/
├── backend/        # Backend Node.js application
└── frontend/       # Frontend React.js application

Steps to Run the Application

1. Clone the Repository
git clone https://github.com/Danishkhan-DK/Squadra.git
cd Squadra

2. Start the Backend
Navigate to the backend folder:
cd backend

Install dependencies:
npm install

Start the server:
node server.js
The backend should now be running on http://localhost:5000 (or the port specified in your code).

3. Start the Frontend
Open a new terminal window and navigate to the frontend folder:
cd ../frontend

Install dependencies:
npm install

Start the React app:
npm start
The frontend should now be running on http://localhost:3000.```
